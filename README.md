# LED Matrix Space Shooter 🚀

A bare-metal embedded C and ARM Assembly game developed for the Xilinx Zynq-7000 SoC. This project drives a DM163 LED matrix directly via memory-mapped registers and utilizes hardware timer interrupts for game logic and screen refreshing. 

Made in a group of two for a course.

## Features ✨
* **Bare-Metal C & Assembly:** No operating system. Direct register manipulation and custom ARM assembly (`blinker.S`) for onboard LED control.
* **Custom LED Matrix Driver:** Implements a driver for the DM163 LED controller, bit-banging the transmission of 144 bits per refresh cycle.
* **Hardware Interrupts:** Utilizes the ARM Generic Interrupt Controller (GIC) and Xilinx Triple Timer Counters (TTC) for decoupled screen rendering (800Hz) and game logic (10Hz).
* **Interactive Gameplay:** Read inputs from GPIO buttons/switches to move the ship, fire bullets, and reset the game state. Includes collision detection and win/loss animations.

## Tech Stack 🛠️
* **Hardware:** Zynq-7000 SoC (ARM Cortex-A9), DM163 8x8 RGB LED Matrix.
* **Software:** Embedded C, ARM Assembly, Xilinx SDK / Vitis.
* **Key Concepts:** Memory-mapped I/O, Interrupt Service Routines (ISRs), Bit-banging, Volatile variables for interrupt safety.

## Project Structure 📂
* `main.c` - Core game loop, state management, and coordinate tracking.
* `Pixel.c` / `Pixel.h` - DM163 LED matrix driver implementation and graphics rendering.
* `Interrupt_setup.c` / `.h` - Configuration of the GIC, TTC timers, and GPIO button interrupts.
* `blinker.S` - ARM Assembly routine for controlling onboard logic LEDs.
* `lscript.ld` - Linker script defining the memory layout for the Zynq PS7 DDR and RAM.

## Getting Started 🏁

### Prerequisites
* Xilinx Vitis or Xilinx SDK installed.
* A Zynq-7000 development board connected via JTAG/UART.

### Build and Run
1. Create a new application project in Xilinx Vitis/SDK.
2. Replace the autogenerated `src` files with the contents of this repository.
3. Build the project (Ctrl+B).
4. Compile the project (Ctrl+B). This generates the .elf binary
5. Connect the board and select Run As -> Launch on Hardware.
6. Vitis will automatically program the FPGA, initialize the system through init_platform(), and start execution.

## Controls 🎮
* **BTN0:** Move Right
* **BTN1:** Move Left
* **BTN2:** Fire
* **SW0:** Reset Game
